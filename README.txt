The Student Voice block is a student-teacher survey tool. It allows students to evaluate teachers based on set questions.

To Install
1. Copy the completion_progress directory to the blocks/ directory of your Moodle instance
2. Visit the notifications page

For more information, visit...
http://docs.moodle.org/en/Installing_contributed_modules_or_plugins

To set up surveys and questions, once the Student Voice block installation has been completed and initial settings are set, return
to the block's global settings page to find a link to the survey config page (/blocks/voice/surveyconfig.php).
